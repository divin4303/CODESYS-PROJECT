Industrial Automation System

CODESYS • Raspberry Pi • Factory I/O • Node-RED

Project Overview

This project demonstrates an industrial automation system using CODESYS PLC programming on a Raspberry Pi, integrated with Factory I/O for process simulation and Node-RED for monitoring and visualization.

System Architecture

  • Raspberry Pi running CODESYS Control
  • Factory I/O for industrial process simulation
  • PLC logic developed using Ladder Logic / Structured Text
  • Node-RED dashboard for real-time monitoring
  • Industrial communication via Modbus TCP / OPC UA

Key Features

  • Real-time PLC-based control system
  • Start/Stop and emergency handling
  • Sensor and actuator monitoring
  • Industrial simulation with Factory I/O
  • Scalable Industry 4.0 architecture

Applications

Smart manufacturing, industrial automation training, digital twin systems, and Industry 4.0 learning environments.

Future Enhancements

  • Cloud integration using MQTT
  • Predictive maintenance features
  • Alarm and fault detection system
  • Physical sensor integration
