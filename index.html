Industrial Automation System

CODESYS • Raspberry Pi • Factory I/O • Node-RED

Project Overview

This project demonstrates an industrial automation system using CODESYS PLC programming on a Raspberry Pi, integrated with Factory I/O for process simulation and Node-RED for monitoring and visualization.

System Architecture

The system consists of a Raspberry Pi running CODESYS Control connected to Factory I/O for industrial process simulation. PLC logic is developed using FBD and Structured Text. Modbus TCP enables communication between CODESYS and Factory I/O, while OPC UA connects CODESYS with Node-RED.

Node-RED Dashboard

The Node-RED dashboard operates as an OPC UA client, enabling secure real-time monitoring and control of PLC variables.

  • Start/Stop control and box counting via PLC registers
  • Live analog monitoring with setpoint adjustment
  • High-speed time-series data visualization

Key Features

  • Real-time PLC-based control
  • Start/Stop and emergency handling
  • Sensor and actuator monitoring
  • Factory I/O industrial simulation
  • Industry 4.0–ready architecture
